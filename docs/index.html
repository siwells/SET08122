Algorithms & Data Structures (SET08122)

A Module taught in the School of Computing @ Edinburgh Napier University

View project on GitHub

Whilst we can write software without paying any real attention to the way we structure our data, or the algorithms that we use to manipulate that data, our programs are likely to be quite poor. For example, they might run slowly, or use a lot of memory, or the problem we are trying to solve might not actually have a solution, or it could take 10,000 years to calculate.

In this module we will study a variety of data structures & algorithms, as well as some related topics such as how to reason about when a data structure is appropriate to use, or whether an algorithm is better (or worse) than another in a give situation.
